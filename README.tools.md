# tools 工具函数

## String,Number 格式化 fromat.tools.js

```js
import {
  enQuantile, // 分位格式化 , 12,345,678
  deQuantile, // 反解分位
  titleCase, // 大驼峰
} from "./fromat.tools.js";
```

## Number 工具函数 Number.tools.js

```js
import {

} from "./Number.tools.js";
```
